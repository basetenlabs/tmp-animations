Baseten animation prototypes

A minimal GitHub Pages wrapper for animated HTML artifacts. Drop your final caching-diagram.html into the repo root and GitHub Pages will serve it as a normal webpage that Notion can link to or attempt to embed.
GitHub Pages ready
/caching-diagram.html

How to use

  1. Replace caching-diagram.html with your final animated HTML.
  2. Push this repo to basetenlabs/temp-animations.
  3. Enable GitHub Pages from the default branch root.

Resulting URLs

Homepage: https://basetenlabs.github.io/temp-animations/
Direct artifact: https://basetenlabs.github.io/temp-animations/caching-diagram.html

Notion tip

In Notion, the direct page URL usually behaves better than a GitHub Releases asset URL because it is served as a normal webpage instead of a downloadable release file.
